🎯 InterviewGen AI

InterviewGen AI is a full-stack MERN application that generates AI-powered mock interview questions and answers based on user skills using Google Gemini API.
It includes authentication, protected routes, interview storage, and a modern UI.


Features

🔐 User Authentication

Signup / Login with JWT & cookies

Protected routes

🤖 AI Mock Interview Generator

Generate interview Q&A by skill (React, Node, MongoDB, etc.)

Powered by Google Gemini AI

💾 MongoDB Storage

Save generated interviews per user

Fetch & delete interviews

🧠 Redux Toolkit

Auth state persistence

🎨 Modern UI

Tailwind CSS

Framer Motion animations

🌐 REST APIs

Secure backend with Express & Mongoose
